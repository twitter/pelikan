Pelikan Docs
============


**Pelikan**  is the unified cache backend framework that provides all executables for accessing and storing cached data. `go/pelikan <http://go/pelikan>`_ points here.

Subscribe to `cache-user <cache-user@twitter.com>`_ for announcement, send feedback and questions to `cache-team <cache-team@twitter>`_.
